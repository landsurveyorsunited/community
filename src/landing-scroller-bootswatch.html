Try It In Action

If you are creating a single-page website with smooth scrolling navigation...

...You Need This Template!

Smooth scrolling navigation is better than jumping over to sections. But NEVER use again jQuery's $.animate() and $.fade() functions. They slow everything else down, including other animation libraries. So we built this template with VelocityJs - incredibly fast, and it features also color animation, transforms, loops, easings, SVG support. It is the best of jQuery and CSS transitions combined.
